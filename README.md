## Installation

To install the `pbcheck` tool, follow these steps:

**Clone the repository:**
```bash
    git clone https://github.com/4LIF27/pbcheck.git
    cd pbcheck
    pip install requests
```

## Usage
To use the `pbcheck` tool, run the following command:
```bash
python pbcheck.py
```
