## Installation

To install the `pbcheck` tool, follow these steps:

**Clone the repository:**
```bash
    git clone https://github.com/4LIF27/pbcheck.git
    cd pbcheck
    pip install requests
```

## Usage
---

▶ Usage

Run the tool:

python pbcheck.py

Prepare your account list inside:

akun.txt

```bash

```
